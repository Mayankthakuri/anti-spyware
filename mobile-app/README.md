# AntiSpyware Pro - Mobile Application

A production-ready cybersecurity mobile application built with React Native, Expo, and TypeScript. Features the same quarantine system and mitigation options as the web version with a native mobile interface.

## 🚀 Features

- **Real-time Threat Monitoring** - Live dashboard with system health metrics
- **Evidence Vault** - Comprehensive threat analysis interface
- **Quarantine System** - Complete isolation and mitigation management
- **Forensic Logs** - Real-time security event streaming
- **System Scanner** - Interactive scanning with progress visualization
- **Native Mobile UI** - Optimized for iOS and Android devices
- **Glassmorphism Design** - Modern cybersecurity aesthetic with blur effects

## 🛠 Tech Stack

- **React Native** - Cross-platform mobile development
- **Expo** - Development platform and tools
- **TypeScript** - Type-safe development
- **Expo Linear Gradient** - Gradient backgrounds
- **Expo Blur** - Glassmorphism blur effects
- **React Native Reanimated** - Smooth animations
- **Expo Vector Icons** - Comprehensive icon library

## 📦 Installation

```bash
# Install Expo CLI globally
npm install -g @expo/cli

# Install dependencies
npm install

# Start development server
npm start

# Run on iOS simulator
npm run ios

# Run on Android emulator
npm run android
```
