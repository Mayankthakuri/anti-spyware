# AntiSpyware Pro - Web Application

A production-ready cybersecurity command center built with React 18, TypeScript, Tailwind CSS 3, and Framer Motion. Features a breathtaking glassmorphism UI inspired by enterprise security dashboards.

## 🚀 Features

- **Real-time Threat Monitoring** - Live dashboard with system health metrics
- **Evidence Vault** - Comprehensive threat analysis and quarantine management
- **Forensic Logs** - Real-time log streaming with severity indicators
- **System Scanner** - Interactive scanning with progress visualization
- **Glassmorphism Design** - Modern cybersecurity aesthetic with smooth animations
- **Responsive Layout** - Optimized for desktop and mobile devices

## 🛠 Tech Stack

- **React 18** - Latest React with concurrent features
- **TypeScript** - Type-safe development
- **Tailwind CSS 3** - Utility-first styling with custom cybersecurity theme
- **Framer Motion** - Smooth animations and transitions
- **Lucide React** - Modern icon library

## 📦 Installation

```bash
# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build
```

## 🎨 Design System

### Color Palette
- **Cyber Blue**: Primary interface elements
- **Threat Colors**: Severity-based color coding (Green/Yellow/Orange/Red)
- **Matrix Green**: Accent color for active states
- **Glass Effects**: Semi-transparent overlays with backdrop blur

### Components
- **GlassCard**: Reusable glassmorphism container
- **ThreatMonitor**: Real-time system status dashboard
- **EvidenceVault**: Threat detection and analysis interface
- **ForensicLogs**: Live log streaming component
- **ScanProgress**: Interactive scanning interface

## 🔧 Configuration

The application uses a custom Tailwind configuration with:
- Extended color palette for cybersecurity themes
- Custom animations (scan, glow, pulse)
- Glassmorphism utilities
- Responsive breakpoints

## 🚀 Deployment

```bash
# Build production bundle
npm run build

# Serve static files
npx serve -s build
```
